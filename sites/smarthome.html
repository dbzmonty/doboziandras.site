Dummy Smart Home


Introduction:

The focus of this project is to create a small smart home which consists of two main parts, a controller and a mobile application.

Operation:

Lights

Thermostat

Alarm

Hardware:

  • ATmega328P microcontroller
  • 1602 LCD module
  • HC-05 bluetooth module
  • HC-SR501 infrared motion sensor
  • LM35 temperature sensor
  • 4 channel relay module
  • Photoresistor
